| 环境变量 | 说明 |
| --- | --- |
| ``CB_HOST`` | Capella host |
| ``CB_USER`` | 用户名 |
| ``CB_PASSWORD`` | 密码 |
| ``CB_BUCKET`` | bucket 名 |
| ``CB_SCOPE`` | scope 名（可选） |
| ``EXPORT_DIR`` | 导出目录，例如 ``/backup/export`` |
| ``EXPORT_FILE`` | NDJSON 文件名，例如 ``export.json`` |
| ``R2_BUCKET`` | R2 bucket 名 |
| ``R2_PATH`` | 上传路径，例如 ``capella-backup/`` |
| ``ARCHIVE_NAME`` | 压缩包名，例如 ``capella_backup_$(date ``+%F).tar.gz`` |
